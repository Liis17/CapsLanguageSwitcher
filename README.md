# CapsLock Language Switcher (WPF)

Небольшое WPF-приложение, которое превращает клавишу `CapsLock` в глобальный переключатель раскладки клавиатуры. После нажатия `CapsLock`:

- Системная раскладка переключается на следующую.
- `CapsLock` сбрасывается в выключенное состояние (лампочка не горит).
- Работает глобально — даже если окно приложения свёрнуто или не в фокусе.
- В трее доступна иконка с контекстным меню.

---

## Возможности

- ✅ Перехват `CapsLock` по всей системе
- ✅ Переключение раскладки клавиатуры
- ✅ Принудительное выключение `CapsLock`
- ✅ Сворачивание в трей и работа в фоне
- ✅ Автозапуск вместе с Windows
- ✅ Локализация интерфейса (русский / английский) с переключением на лету
- ✅ Выбор метода переключения раскладки (`PostMessage` или симуляция `Win+Space`)
- ✅ Защита от запуска нескольких копий — повторный запуск активирует уже работающий экземпляр
- ✅ Сохранение пользовательских настроек между запусками
- ✅ Горячие клавиши для полного выхода: `Ctrl+Q` или `Shift` + клик по крестику

---

## Настройки

Все настройки сохраняются в `%AppData%\CapsLanguageSwitcher\settings.json` и применяются автоматически при следующем запуске.

| Настройка | Значения | Описание |
|---|---|---|
| Метод переключения | `PostMessage` / `SendInput` | Способ смены раскладки. `PostMessage` — классический, `SendInput` — симуляция `Win+Space`, работает в большем числе приложений |
| Язык интерфейса | `ru` / `en` | Язык всех элементов UI и трея, переключается без перезапуска |
| Автозапуск | вкл / выкл | Регистрируется в `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` |

---

## Примечания

- Окна, запущенные **от имени администратора**, могут блокировать перехват `CapsLock`. Чтобы переключение работало в таких окнах, запускайте приложение с правами администратора. Альтернативный метод переключения этого ограничения не обходит.
- При сбое установки низкоуровневого хука клавиатуры пользователь получит уведомление; приложение продолжит работать без переключения раскладки до перезапуска.
- Целевая платформа — .NET 10 (Windows). Используется WPF + `Windows Forms` для `NotifyIcon`.

---

## Сборка

```powershell
dotnet build CapsLanguageSwitcher.sln -c Release
```

Опубликованная сборка лежит в `CapsLanguageSwitcher\bin\Release\net10.0-windows10.0.26100.0\`.
